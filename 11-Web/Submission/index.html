Objective: Summary of Latitude vs X

This analysis looks at the correlation between location and weather changes. Using data from OpenWeatherApi, latitude was plotted in scatter charts against: Max Temperature, Humidity, Cloudiness, and Wind Speed (mph) for 500 randomly selected cities.
