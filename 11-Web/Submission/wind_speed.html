Latitude vs. Wind Speed


This plot correlates wind speed (mph) at each of the latitides of the data set. There is no apparant correlation, but the data does cluster between 0 and 20 mph windspeed globally. There is one outlier location recorded in the data set close to the North Pole that has a recorded windspeed of over 40 mph.
